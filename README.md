# Pusheen

Cat Pusheen on your screen

![](demo.png)

## Getting Started

### Install

```sh
curl https://raw.githubusercontent.com/BookGin/pusheen/master/pusheen >> ~/.bashrc
source ~/.bashrc
```

If you use `zsh`, please specify both lines below in `~/.zshrc`.
